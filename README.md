# Esencia y Taza — Prototipo de landing

Prototipo de demostración de la landing page de **Esencia y Taza**, café de
origen de Santa Rosa de Osos, Antioquia (Colombia).

> ⚠️ **Es un prototipo.** Imágenes, precios y número de contacto son
> provisionales, para revisión interna del equipo. No es la versión final.

## Ver el prototipo

Abre `landing-prototipo.html` en cualquier navegador (doble clic). Es un
único archivo autónomo, sin dependencias.

## Estructura

- `landing-prototipo.html` — la landing completa (HTML + CSS + ilustraciones SVG en un solo archivo).

## Estado

Maqueta para alinear diseño, estructura y textos con el equipo antes de
construir la versión final con fotos y datos reales.
