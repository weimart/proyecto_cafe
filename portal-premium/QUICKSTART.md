# ⚡ INICIO RÁPIDO EN 5 MINUTOS

## Paso 1: Abrir en VS Code (30 segundos)

```bash
# Abre la terminal en la carpeta PROJECT_VSCODE y escribe:
code .

# O: Drag & drop la carpeta a VS Code
```

## Paso 2: Instalar extensión Live Server (1 minuto)

1. Presiona `Ctrl+Shift+X` (Extensions)
2. Busca: `Live Server`
3. Haz click en "Install"

## Paso 3: Iniciar servidor local (30 segundos)

1. Click derecho en `src/index.html`
2. Selecciona: `Open with Live Server`

**¡Se abre automáticamente en tu navegador!** 🎉

## Paso 4: Editar contenido (2 minutos)

En VS Code, abre `src/index.html` y cambia:

```html
<h1>Del Árbol a tu Taza</h1>
<!-- Cambia a: -->
<h1>Tu Nombre Aquí</h1>
```

**El navegador se actualiza automáticamente** ✨

## Paso 5: Personalizar estilos (1 minuto)

Abre `src/assets/css/styles.css` y busca:

```css
:root {
  --color-dark: #3D2817;    /* Tu color aquí */
  --color-gold: #D4AF37;    /* Tu color aquí */
}
```

## 🎨 Cambios Rápidos

### Cambiar Colores
Edita `:root` en `styles.css`

### Cambiar Textos
Edita el contenido en `index.html`

### Agregar Imágenes
1. Copia imagen a `src/assets/images/`
2. En HTML: `<img src="./assets/images/tu-foto.jpg">`

### Agregar Nuevas Secciones
Copia/pega una `<section>` existente en `index.html`

## 🆘 Ayuda Rápida

**¿No se actualiza la página?**
- Presiona F5 para refrescar
- O Ctrl+Shift+R (limpiar caché)

**¿Live Server no abre?**
- Instala extensión "Live Server"
- Click derecho en index.html → Open with Live Server

**¿No ves cambios de CSS?**
- Guarda el archivo (Ctrl+S)
- Espera 1 segundo
- La página se actualiza automáticamente
