🎯 FLUJO DE TRABAJO TÍPICO

1. ABRIR PROYECTO
   VS Code → Ctrl+K Ctrl+O → PROJECT_VSCODE → Abrir

2. INSTALAR EXTENSIONES
   VS Code → Ctrl+Shift+X → Live Server → Instalar

3. INICIAR SERVIDOR
   src/index.html → Click derecho → Open with Live Server

4. EDITAR CONTENIDO
   index.html → Cambia textos → Ctrl+S → Navegador se actualiza

5. CAMBIAR ESTILOS
   styles.css → Edita colores/tamaños → Ctrl+S → Navegador se actualiza

6. AGREGAR IMÁGENES
   images/ → Copia fotos → index.html → Actualiza src="" → Ctrl+S

7. PROBAR EN MÓVIL
   F12 en navegador → Toggle device toolbar → Prueba responsividad

8. PUBLICAR (Cuando esté listo)
   Vercel / Netlify / GitHub Pages → Deploy
